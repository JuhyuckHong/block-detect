# Edge ROI Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace the old shifted lower-left ROI with a fixed lower-left frame-edge ROI from left-edge 30% down to bottom-edge 40% across.

**Architecture:** `BlackPixelClassifier` owns the ROI geometry and score calculation. `Settings`, `DetectionPipeline`, and `DetectionGui` pass and persist the two new endpoint ratios so classification reports and preview overlays match the actual classifier behavior.

**Tech Stack:** Python 3.11+, Pillow, unittest.
